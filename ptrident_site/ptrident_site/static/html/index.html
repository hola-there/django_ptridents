Poseidon's Tridents

A Mesh-Networked set of sensors that collect data
during natural disastors and more!

READ MORE CONTACT US

Innovative Tech For Natural Disasters

Provides communities with a low cost, modular (open-source optional)  tools and software to be able to see data in real-time relevent to the situation on the ground.

Modern Technologies

Built with open-source tech!:
Python 3, Django, C, etc.

Scalable

Add a few nodes or even more! Connect as many data points as needed and get what you need. Then connect to other cloud services to get even more functionality.

Multifunctional

Master & Slave node transfer data as needed whilst providing an ad-hoc network for those in the areas encompassing the nodes!

Modular

Can be used in natural disaster situations to collect data and provide an ad-hoc wifi network for local communications and data viewing. Also as a cheap air pollution sensors, local weather station, etc.

Open Source Options

We utilize Django, Python, PlatformIO, Raspbery Pi, Arduino, Github, C, C++, Adafruit, etc.

Multifunctional

Possibly industrial versions could be made later on, but for now you add as many (off-the-shelf) sensors as needed and use whichever materials you'd like for the pipe.

Secure

Utilizes a local ssl certificate created from Let's Encrypt, https, WPA2 wifi network, UART & JTAG pinouts aren't exposed, etc.

Our Team

We cut and drilled a pvc pipe, we soldered components for our arduino and raspberry pi boards, we created a web server, we created a hidden ctf, we created a website, we wrote C, Python, HTML, CSS, Javascript, and more in order to get all of this to work just for a prototype. It could be much more and be quite useful for a variety of situations.
